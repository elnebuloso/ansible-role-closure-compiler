# Ansible Role - closure-compiler for Linux Server

## Requirements

This role requires Ansible 2.0 or higher, and platform requirements are listed in the metadata file.

## Supported Distributions

- ubuntu14
- ubuntu16

## Role Variables

None.

## Example Playbook

```

  roles:
    - role: elnebuloso.closure-compiler
```
